## データストア現況 (v1.0)

| ストア | 状態 |
|--------|------|
| VSAM KSDS (KYKMAST/KENFILE/RYOFILE) | 現行主力. CICS/バッチが実走で使用 |
| IMS/DB (MTRDBD 計器設備DB + DMDDBD) | **計器資産の正本 (System of Record)**. 乗率・検定満期 (和暦 E-01)・機種・取付履歴 (階層ツイン) は IMS にのみ存在し、日次サイクル先頭の D005 (RBMTR00C, DL/I GU/GN/GNP) が計器マスタ順次 MTRMAST を抽出する。後続の検定監視・料金系はすべてこの抽出物を読む — 階層DBが業務の背骨として現役。DMDDBD (デマンド当日値) は資産定義として現存 |
| Db2 (RACS.DMD_RIREKI) | RAAUP00C が EXEC SQL INSERT で実走使用 |
| 順次/テープ系 | 受信 VB・全銀・請求媒体・世代管理ほかバッチ 19 本が実走使用 |
